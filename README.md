## Build HelloWorld_CPP
To build the C++ project enter into HelloWorld_CPP folder and execute the following commands:
```shell
$ mkdir build
$ cd build
$ cmake ../ [extra options]
```

## Runing HelloWorld_PY
```shell
$ cd HelloWorld_PY
$ conda env create
$ activate hw_env
$ python HelloWorld.py
```
